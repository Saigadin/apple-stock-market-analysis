Apple Stock Market Analysis

Project Overview

This project performs an end-to-end analysis of Apple (AAPL) stock market data using **Python, SQL, and Power BI**.
The objective is to explore historical stock data, identify trends and patterns, and build an interactive dashboard that provides insights into stock performance and market behavior.

The project demonstrates a complete **data analyst workflow**, including data cleaning, analysis, visualization, and business insights.

---

Tools & Technologies

Python – Data cleaning and exploratory analysis
SQL (MySQL) – Data querying and aggregation
Power BI – Interactive dashboard and visualization
DAX – Calculated columns and measures for analysis

---

Project Workflow

1. Data Collection

    Historical Apple stock data was collected and stored in CSV format.

2. Data Cleaning (Python)

    Removed missing values

   Created new features:
     Daily Return
     30-Day Moving Average (MA30)
     Price Change

4. Data Analysis (SQL)

    Calculated average closing price
    Identified highest trading volume days
    Analyzed bullish vs bearish market behavior
    Measured price volatility

5. Data Visualization (Power BI)

     Built an interactive dashboard including:

     KPI cards
     Stock price trend chart
     Moving average analysis
     Trading volume analysis
     Market sentiment visualization

---

 Dashboard Features

The Power BI dashboard includes:

  Stock Price Trend Analysis
  Moving Average Comparison
  Trading Volume Analysis
  Bullish vs Bearish Market Sentiment
  Key Performance Indicators (KPIs)

---

 Key Insights

 Apple stock shows a **long-term upward trend with occasional volatility.
 Trading volume spikes** often coincide with major price movements.
 Bullish trading days slightly outnumber bearish days, indicating positive investor sentiment.
 The 30-day moving average , helps identify longer-term stock market trends.
